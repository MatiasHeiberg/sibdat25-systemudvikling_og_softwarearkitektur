# Udfordringer vi har haft med vandfaldsmodellen?
- Scopecreep
- Vi havde ikke nogen feedback eller anden kilde til krav, så vi opfandt flere problemer end nødvendig, fordi vi kunne ikke vurdere om de var ægte eller tænkte.
# Hvilke scenarier vil vandfaldsmodellen være god?

- Business as usual projekter
- Alle projekter hvor analysebyrden er lav
- Få ukendte variabler (unknown unknowns)
