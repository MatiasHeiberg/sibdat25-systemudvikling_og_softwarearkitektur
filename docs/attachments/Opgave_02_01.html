Dag 02 — Softwarearktitektur

02_01 — Encapsulation

Udtræk logik i selvstændige klasser

Problemet

I 02_01_A er al rabatlogik gemt inde i Checkout.GetTotal() via if-else. Checkout kender alle rabattyper (NONE, PERCENT10, FIXED50) og beregner dem selv. Problemet: Hver gang en ny rabattype tilføjes, skal Checkout ændres.

Før (02_01_A)
GetTotal(subtotal, discountType)
  if discountType == "PERCENT10"
    discount = subtotal * 0.10
  else if discountType == "FIXED50"
    discount = 50
  else
    discount = 0
  return subtotal - discount
Efter (02_01_B)
GetTotal(subtotal)
  discount = _discountCalculator
    .CalculateDiscount(subtotal)
  return subtotal - discount

Løsningen: Encapsulation

En DiscountCalculator-klasse udtrækker rabatlogikken. Checkout kender ikke længere detaljerne — den delegerer beregningen til DiscountCalculator. Checkout modtager DiscountCalculator via constructor injection.

Grøn Refaktorér rabatlogikken

Tag udgangspunkt i Exercise_02_01_A_UdenEncapsulation.

1 — Opret DiscountCalculator

Opret klassen DiscountCalculator med en constructor der tager rabattypen (string) og en metode CalculateDiscount(subtotal) der returnerer rabatbeløbet. Støttede typer: "NONE" (0 kr.), "PERCENT10" (10 %) og "FIXED50" (50 kr.).

2 — Opdater Checkout

Fjern if-else logikken fra GetTotal. Tilføj et felt _discountCalculator og modtag en DiscountCalculator via constructor. GetTotal delegerer til DiscountCalculator.

3 — Opdater Program.cs

Opret en DiscountCalculator med den ønskede rabattype og injicer den i Checkout.

Acceptkriterier

  1. DiscountCalculator indeholder al rabatlogik
  2. Checkout.GetTotal tager kun subtotal — ingen discountType parameter
  3. Programmet kompilerer og giver samme output som 02_01_A
Gul Test DiscountCalculator og Checkout

Opret et testprojekt Exercise_02_01_B_MedEncapsulation.Tests og skriv 6 unit tests.

Test DiscountCalculator

  1. NONE giver 0 kr. rabat for subtotal 500 kr.
  2. PERCENT10 giver 50 kr. rabat for subtotal 500 kr.
  3. FIXED50 giver 50 kr. rabat for subtotal 500 kr.

Test Checkout

  1. GetTotal uden rabat returnerer fuld pris (500 kr.)
  2. GetTotal med PERCENT10 returnerer 450 kr.
  3. GetTotal med FIXED50 returnerer 450 kr.
